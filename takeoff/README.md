# Takeoff 起飞模块

采用原版xray内核，支持多种隧道模式。

## 手动修改服务端地址和协议

运行init.sh脚本初始化ip地址

```bash
./init.sh run <YOUR-SERVER-IP-ADDRESS>
./init.sh run 11.0.1.10
```



## 启动

```bash

git clone https://github.com/Paper-Dragon/tunnel-through-the-wall.git && cd tunnel-through-the-wall/takeoff && docker-compose up -d
```
